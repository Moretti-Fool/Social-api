Social Media API Documentation

Built with FastAPI, PostgreSQL and Docker

Posts Management

  • Retrieve Posts: Fetch a list of posts with optional filters (limit, skip, and keyword search).
  • Create Posts: Add new posts with fields like title, content, and publication status.
  • Get Specific Post: Fetch detailed information about a single post using its ID.
  • Update Posts: Modify existing posts (e.g., edit title, content, or publication status).
  • Delete Posts: Remove posts permanently by their ID.

Users Management

  • User Registration: Create new users by providing email and password.
  • Retrieve User Details: View information for a specific user.
  • User-Specific Posts: Fetch all posts created by a particular user.

Authentication

  • User Login: Authenticate via username/password, receiving an access token for future requests.

Voting System

  • Vote on Posts: Users can cast votes on posts, with support for specifying the vote direction (e.g., upvote/downvote).
