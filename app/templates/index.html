Welcome to the FastAPI-powered API

This API provides endpoints for managing posts, users, and voting functionalities. Below are the main features:

Posts

  • Get Posts: Retrieve a list of posts with options to limit the results, skip entries, and search by keyword.
  • Create Post: Add a new post with a title, content, and publication status.
  • Get Post by ID: Retrieve details of a specific post.
  • Update Post: Modify an existing post's title, content, or publication status.
  • Delete Post: Remove a specific post by ID.

Users

  • Create User: Register a new user by providing an email and password.
  • Get User: View details of a specific user.
  • Get User's Posts: Retrieve all posts created by a specific user.

Authentication

  • Login: Users can authenticate via username and password, receiving an access token for future requests.

Voting

  • Vote: Users can vote on posts, where the vote direction is specified.
